The Fylo Landing Page

Project Description

In a bid to challenge my skill and become better in creating single page landing pages, I created the Fylo Landing Page from frontendmentor.io.

The project was quite challenging as I didn't have a ui/ux design file to work with so I have to use the design images provided. It wasn't really easy ensuring that the final page looked exactly like the actual one both on mobile and desktop.

Nonethless, I am happy to say that the effort was worth it as I learned a lot about flex display property which I found out was quite awesome to work with.
